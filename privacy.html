Overview

What This Site Collects

FederalTaxRules.com is a static website. It has no user accounts, no login system, no contact forms that submit data anywhere, and no server-side processing. Visitors browse reference articles and are not asked to provide any personal information as a condition of using the site.

This site does not knowingly collect, store, or process personally identifiable information from visitors.

Infrastructure

Hosting and Delivery

This site is hosted as a static site on a third-party hosting infrastructure. Standard server-side logs may record technical request data such as IP addresses, browser type, referring URLs, and page access timestamps as part of normal web server operation. This logging is managed by the hosting provider and is not used by this site for any analytical or marketing purpose.

These logs are a standard part of how web hosting infrastructure functions and are not controlled by this site directly.

This site does not control which ads Google displays or how Google's advertising systems use cookie data. Google's privacy practices are governed by Google's own privacy policy.
